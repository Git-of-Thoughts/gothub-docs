# Introduction

##

### To start a new chat

This will clear chat history but not system message.

```python
as new chat
What is the meaning of life?
```

### To execute Python code

```python
as code
print("Hello world!")
```

### To talk to GPT-3.5 in current cell

```python
with gpt-3.5
What is the meaning of life?
```

or

```python
with gpt3.5
What is the meaning of life?
```

### To talk to GPT-4 in current cell

```python
with gpt-4
What is the meaning of life?
```

or

```python
with gpt4
What is the meaning of life?
```

## As code: `got` library

### To talk to the default large langauge model

```python
as code
from got import ask
ask("What is the meaning of life?")
```

### To talk to GPT-3.5 in current cell using code

```python
with gpt-3.5 as code
from got import ask
ask("What is the meaning of life?")
```

### To talk to GPT-4 in current cell using code

```python
with gpt-4 as code
from got import ask
ask("What is the meaning of life?")
```
